# Analyzing CO₂ Emissions in Environmental Experiments

## Overview

In this project, I analyzed data from an environmental experiment to investigate how different fuel sources influenced CO₂ emissions across multiple geographical regions. I identified the experimental design, visualized emission patterns, and applied statistical hypothesis testing to determine whether differences in emissions between fuel sources were statistically significant.

## Key Insights

* Identified the experimental setup as a randomized block design.
* Compared CO₂ emissions across different fuel sources within each geographical region.
* Visualized emission distributions using box plots to highlight regional differences.
* Performed one-way ANOVA to test for significant differences in CO₂ emissions among fuel sources.
* Applied Bonferroni correction for multiple pairwise comparisons to identify statistically significant differences while controlling for false positives.

## Skills Demonstrated

* Data visualization with Matplotlib and Seaborn
* Experimental design interpretation
* Hypothesis testing using ANOVA
* Multiple comparison testing with Bonferroni correction
