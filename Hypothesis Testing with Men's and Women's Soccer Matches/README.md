# Comparing Goal Scoring in FIFA World Cup Matches

## Overview

In this project, I analyzed international FIFA World Cup match data to compare the number of goals scored in men's and women's tournaments. After preparing and filtering the datasets, I examined the goal distributions and performed a statistical hypothesis test to determine whether women's World Cup matches tend to have a higher number of goals than men's matches.

## Key Insights

* Filtered the datasets to include FIFA World Cup matches played since 2002.
* Calculated the total number of goals scored in each match.
* Examined the distribution of goals using histograms.
* Applied the Mann–Whitney U test to compare goal totals between men's and women's tournaments.
* Determined whether there was sufficient statistical evidence to support the hypothesis that women's matches have a higher average number of goals.

## Skills Demonstrated

* Data preprocessing with Pandas
* Data visualization with Matplotlib
* Statistical hypothesis testing
* Mann–Whitney U test
